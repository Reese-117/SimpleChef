# Instructions
Students are required to submit their project code for this assignment. You may either upload all relevant code files directly to Canvas or submit a document containing a working link to your GitHub repository. Make sure your repository is public.  Your submission must include a **README** file that explains the project structure, major components, libraries/frameworks used, and instructions for running the code. The code should demonstrate good software engineering practices such as modular design, meaningful naming, consistent formatting, and appropriate comments.

## Grading Criteria (10 Points)
1.  Code Organization & Readability (4)
2.  Software Engineering Practices (3)
3.  Documentation & Code Understanding (3)
